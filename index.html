1. Introduction

Le modèle Simandré permet de collecter des informations météorologiques telles que la température, la pression d'air et la qualité de l'air au sein de votre entreprise. Ce guide vous aidera à comprendre comment le configurer et l'utiliser.

2. Installation et configuration

Pour installer et configurer le module météo, veuillez suivre les étapes suivantes :

  1. Choisissez un endroit ouvert, idéalement le centre d'un open-space.
  2. Branchez le module à une prise de courant.
  3. A la première utilisation :
    • Connetez-vous en Bluetooth au module se nommant "ESP_meteo".
    • Rentrez les informations de votre Wifi (le nom et mot de passe).
    • Le module se connectera alors à votre Wifi et se déconnectera de votre Bluetooth.
  4. Votre module est opérationnel.

3. Utilisation

Pour utiliser le module météo, veuillez suivre les étapes suivantes :

3.1 Collecte de données

  • Les données récoltées sont envoyées directement sur nos serveurs, vous n'avez pas besoin d'intervenir.

3.2 Affichage des données

  1. Rendez-vous sur notre site www.datasense-solutions/simandré, connectez-vous à l'aide de vos identifiants et mot de passe fournis dans l'email de bienvenue.
  2. Une fois connecté, vous pourrez naviguer sur le site qui contient les données collectées de votre module Simandré.

4. Fonctionnement du site internet

Le site internet Simandré dispose de plusieurs fonctionnalités telles que :

  • Un dashboard affichant des indicateurs relatifs à l'ensemble de vos modules Simandré et un plan détaillé des locaux de votre entreprise.
  • L'onglet "Capteur" affiche l'ensemble de vos modules Simandré de façon détaillées.
  • L'onglet "Plan" permet de déposer le plan de votre entreprise. Il vous servira de support dans le placement de vos modules Simandré, symbolisé par des pastilles rouges.
  • En pied de page, vous retrouverez de nombreux liens tel que nos réseaux, nos sponsors, notre mentions légales, nos termes de ventes, notre politique de confidentialité et notre contact.

5. Résolution des problèmes

Si vous rencontrez des problèmes avec le module Simandré, veuillez suivre les étapes suivantes pour les résoudre :

  1. Vérifiez la dernière date de reçu des données. Vous pouvez consulter cette informations dans le rubrique "Capteur" de votre session.
    • Si la date est inférieure à 15min par rapport à l'heure courant, cela signifie que le module fonctionne correctement.
    • Sinon, cela signifie que le module ne fonctionne pas correctement. Dans ce cas, passez à l'étape suivante.
  2. Vérifiez le branchement du module. Si celui-ci à subit une perte d'alimentation, une re-configuration du Wifi est nécessaire.
  3. Si il n'y a pas eu de problème d'alimentation, débranchez le module et recommencer l'initialisation.
  4. Si le problème persiste, contactez le support technique de Datasense Solutions.

6. FAQ

Voici quelques questions fréquemment posées sur le module météo :

  1. Q : Combien de modules faudrait-il idéalement ?

    2. R : Tout dépend de votre surface. Mais nous vous conseillons de mettre un module par zone de 50 m². L'emplacement idéale ce trouve être le milieu d'un open-space.

  2. Q : Puis-je personnaliser la couleur des modules ?

    3. R : Oui, vous pouvez personnaliser la couleur des moduleslors de votre commande.

  3. Q : Le module météo est-il sécurisé ?

    4. R : Oui, le module météo utilise des protocoles de sécurité avancés pour protéger vos données.
